11.16 emacspeak-alsaplayer

Defines a simple derived mode for interacting with alsaplayer. alsaplayer navigation commands work via single keystrokes.

11.16.1 emacspeak-alsaplayer Commands

Command: emacspeak-alsaplayer-add-to-queue (resource)

Add specified resource to queue.

Command: emacspeak-alsaplayer-amark-add (name &optional prompt-position)

Set AMark ‘name’ at current position in current audio stream. Interactive prefix arg prompts for position. As the default, use current position.

Command: emacspeak-alsaplayer-amark-jump

Jump to specified AMark.

Command: emacspeak-alsaplayer-backward-10-seconds

Skip backward by 10 seconds.

Command: emacspeak-alsaplayer-backward-minute (minutes)

Skip backwards by minutes.

Command: emacspeak-alsaplayer-backward-ten-minutes (minutes)

Skip backwards by chunks of minutes.

Command: emacspeak-alsaplayer-cd (directory)

Change default directory, and silence its pronunciation.

Command: emacspeak-alsaplayer-clear

Clear or resume alsaplayer

Command: emacspeak-alsaplayer-clip (path start end)

Invoke mp3splt to clip selected range.

Command: emacspeak-alsaplayer-find-and-add-to-queue (pattern)

Find specified resource and add to queue.

Command: emacspeak-alsaplayer-forward-10-seconds

Skip forward by seconds.

Command: emacspeak-alsaplayer-forward-minute (minutes)

Skip forward by minutes.

Command: emacspeak-alsaplayer-forward-ten-minutes (minutes)

Skip forward by chunks of ten minutes.

Command: emacspeak-alsaplayer-info

Speak current path and copy it to kill ring.

Command: emacspeak-alsaplayer-jump (track)

Jump to specified track.

Command: emacspeak-alsaplayer-launch

Launch Alsaplayer. user is placed in a buffer associated with the newly created Alsaplayer session.

Command: emacspeak-alsaplayer-mark-position

Mark currently displayed position.

Command: emacspeak-alsaplayer-mode

Major mode for alsaplayer interaction.

11.16.2 emacspeak-alsaplayer Options

Variable: User Option emacspeak-alsaplayer-auditory-feedback

Turn this on if you want spoken feedback and auditory icons from alsaplayer.

Variable: User Option emacspeak-alsaplayer-device

Device to use for alsaplayer

Variable: User Option emacspeak-alsaplayer-media-directory

Directory to look for media files.

Variable: User Option emacspeak-alsaplayer-program

Alsaplayer executable.
