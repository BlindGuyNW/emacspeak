12.100 emacspeak-indium

INDIUM == Javascript IDE This module speech-enables Indium.

12.100.1 emacspeak-indium Options

Variable: User Option emacspeak-indium-breakpoint-personality

Personality used for indium-breakpoint-face This personality uses voice-lighten whose effect can be changed globally by customizing voice-lighten-settings.

Variable: User Option emacspeak-indium-button-personality

Personality used for indium-button-face This personality uses voice-bolden-medium whose effect can be changed globally by customizing voice-bolden-medium-settings.

Variable: User Option emacspeak-indium-frame-url-personality

Personality used for indium-frame-url-face This personality uses voice-animate whose effect can be changed globally by customizing voice-animate-settings.

Variable: User Option emacspeak-indium-header-personality

Personality used for indium-header-face This personality uses voice-smoothen whose effect can be changed globally by customizing voice-smoothen-settings.

Variable: User Option emacspeak-indium-highlight-personality

Personality used for indium-highlight-face This personality uses voice-animate whose effect can be changed globally by customizing voice-animate-settings.

Variable: User Option emacspeak-indium-keyword-personality

Personality used for indium-keyword-face This personality uses voice-animate whose effect can be changed globally by customizing voice-animate-settings.

Variable: User Option emacspeak-indium-link-personality

Personality used for indium-link-face This personality uses voice-bolden whose effect can be changed globally by customizing voice-bolden-settings.

Variable: User Option emacspeak-indium-litable-personality

Personality used for indium-litable-face This personality uses voice-lighten whose effect can be changed globally by customizing voice-lighten-settings.

Variable: User Option emacspeak-indium-repl-error-personality

Personality used for indium-repl-error-face This personality uses voice-animate-extra whose effect can be changed globally by customizing voice-animate-extra-settings.

Variable: User Option emacspeak-indium-repl-prompt-personality

Personality used for indium-repl-prompt-face This personality uses voice-annotate whose effect can be changed globally by customizing voice-annotate-settings.

Variable: User Option emacspeak-indium-repl-stdout-personality

Personality used for indium-repl-stdout-face This personality uses voice-monotone whose effect can be changed globally by customizing voice-monotone-settings.
