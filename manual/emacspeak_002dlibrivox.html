11.105 emacspeak-librivox

LIBRIVOX == http://www.librivox.org Free Audio Books API Info: https://librivox.org/api/info It provides a simple Web API This module implements an Emacspeak Librivox client.

11.105.1 emacspeak-librivox Commands

11.105.1.1 emacspeak-librivox

Command: emacspeak-librivox (search-type)
Launch a Librivox Search.

11.105.1.2 emacspeak-librivox-play

Command: emacspeak-librivox-play (rss-url)
Play book stream

11.105.1.3 emacspeak-librivox-search-by-author

Command: emacspeak-librivox-search-by-author (author)
Search by author.
Both exact and partial matches for “author”.

11.105.1.4 emacspeak-librivox-search-by-genre

Command: emacspeak-librivox-search-by-genre (genre)
Search by genre.
Both exact and partial matches for “genre”.

11.105.1.5 emacspeak-librivox-search-by-title

Command: emacspeak-librivox-search-by-title (title)
Search by title.
Both exact and partial matches for “title”.

11.105.2 emacspeak-librivox Options

Variable: User Option emacspeak-librivox-local-cache

Location where we cache LIBRIVOX playlists.
