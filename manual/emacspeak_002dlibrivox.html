11.105 emacspeak-librivox

LIBRIVOX == http://www.librivox.org Free Audio Books API Info: https://librivox.org/api/info It provides a simple Web API This module implements an Emacspeak Librivox client.

11.105.1 emacspeak-librivox Commands

Command: emacspeak-librivox-search-by-author (author)

Search by author. Both exact and partial matches for ‘author’.

Command: emacspeak-librivox-search-by-genre (genre)

Search by genre. Both exact and partial matches for ‘genre’.

Command: emacspeak-librivox-search-by-title (title)

Search by title. Both exact and partial matches for ‘title’.
