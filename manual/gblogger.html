11.196 gblogger

This module is obsolete pending implementation of the new Blogger authentication API. That API presently requires one to get an auth token through a heavy-weight Web browser like Chrome or Firefox, and is consequently beyond Emacs.

Simple wrapper using Curl to post/edit Blogger.com Blogs posts are edited as XML nxml-mode is highly recommend and will be used if available.

11.196.1 gblogger Commands

11.196.1.1 gblogger-add-label

Command: gblogger-add-label (label)
Adds labels to gblogger entry being edited.

11.196.1.2 gblogger-atom-display

Command: gblogger-atom-display (feed-url)
Retrieve and display specified feed after authenticating.

11.196.1.3 gblogger-blog

Command: gblogger-blog
Retrieve and display feed of feeds after authenticating.

11.196.1.4 gblogger-delete-entry

Command: gblogger-delete-entry (edit-url)
Delete item at specified edit URL.

11.196.1.5 gblogger-edit-entry

Command: gblogger-edit-entry (url)
Retrieve entry and prepare it for editing.
The retrieved entry is placed in a buffer ready for editing.
“url” is the URL of the entry.

11.196.1.6 gblogger-new-entry

Command: gblogger-new-entry (url &optional title text)
Create a new Blog post.

11.196.1.7 gblogger-sign-in

Command: gblogger-sign-in
Resets client so you can start with a different userid.

11.196.1.8 gblogger-sign-out

Command: gblogger-sign-out
Resets client so you can start with a different userid.

11.196.2 gblogger Options

Variable: User Option gblogger-author

Author name under which we post.

Variable: User Option gblogger-posting-url

URL for posting to this post. You can get the URL from a [post] link shown by “gblogger-blog”

Variable: User Option gblogger-user-email

Mail address that identifies blogger user.

Variable: User Option gblogger-user-password

Password for authenticating to reader account.
