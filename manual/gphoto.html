11.196 gphoto

Emacs client for accessing Picasa — Google Photo Albums See http://code.google.com/apis/picasaweb/overview.html

11.196.1 gphoto Commands

Command: gphoto-album-create

Create a new GPhoto album.

Command: gphoto-albums (&optional prompt)

Display feed of albums. Interactive prefix arg prompts for userid whose albums we request.

Command: gphoto-comment-or-tag (type resource)

Add comments or tags to an existing photo.

Command: gphoto-community-search (query)

Search all public photos.

Command: gphoto-delete-item (url)

Delete specified item.

Command: gphoto-directory-add-photos (directory album-name)

Add all jpeg files in a directory to specified album.

Command: gphoto-download (resource)

Download resource after authenticating.

Command: gphoto-edit-entry (url)

Retrieve metadata for entry and prepare it for editting. The retrieved entry is placed in a buffer ready for editing. ‘url’ is the URL of the entry.

Command: gphoto-feeds (kind user)

Retrieve and display feed of albums or tags after authenticating.

Command: gphoto-photo-add (album-name photo)

Add a photo to an existing album.

Command: gphoto-recent (user kind)

Retrieve feed of recently uploaded photos or comments.

Command: gphoto-sign-in

Resets client so you can start with a different userid.

Command: gphoto-sign-out

Resets client so you can start with a different userid.

Command: gphoto-tags

View feed of tags.

Command: gphoto-user-search (user query)

Retrieve feed of recently uploaded comments for specified user.

Command: gphoto-user-tagsearch (user tag)

Retrieve feed o matches comments for specified user.

Command: gphoto-view (resource)

Retrieve and display resource after authenticating.

11.196.2 gphoto Options

Variable: User Option gphoto-album-default-access

Default access mode for newly created albums.

Variable: User Option gphoto-album-default-commenting-enabled

Default commenting mode for newly created albums.

Variable: User Option gphoto-user-email

Mail address that identifies Picasa user.

Variable: User Option gphoto-user-password

Password for authenticating to Picasa account.
