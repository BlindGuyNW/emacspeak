11.212 soundscape

http://boodler.org is a Python-based SoundScape generator. This module defines Emacspeak conveniences for running Soundscapes. Main Entry Points:

  • M-x soundscape — runs a named SoundScape
  • M-x soundscape-toggle — Enables or disables automatic SoundScapes.

When automatic Soundscapes are enabled, SoundScapes are automatically started and stopped based on the current major mode. Thus, SoundScapes can be thought of as reflecting the mood of the current mode.

11.212.1 soundscape Commands

11.212.1.1 soundscape

Command: soundscape (scape)
Play soundscape.

11.212.1.2 soundscape-display

Command: soundscape-display
Display names of running scapes.

11.212.1.3 soundscape-kill

Command: soundscape-kill
Stop all running soundscapes.

11.212.1.4 soundscape-stop

Command: soundscape-stop (scape)
Stop running Soundscape.

11.212.1.5 soundscape-toggle

Command: soundscape-toggle
Toggle automatic SoundScapes.
When turned on, Soundscapes are automatically run based on current major mode.
