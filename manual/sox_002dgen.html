12.218 sox-gen

Contains a collection of functions that generate sound using SoX. These functions are primarily for use from other Emacs/Emacspeak modules. This module can be used independent of Emacspeak.

12.218.1 sox-gen Commands

12.218.1.1 sox-beats-binaural

Command: sox-beats-binaural (length beat-spec-list gain)
Play binaural audio with beat-spec specifying the various tones.
Param ‘beat-spec’ is a list of ‘(carrier beat) tupples.

(fn LENGTH BEAT-SPEC-LIST GAIN)

12.218.1.2 sox-binaural

Command: sox-binaural (name duration)
C-, b
C-x @ a b
Play specified binaural effect.

(fn NAME DURATION)

12.218.1.3 sox-chakras

Command: sox-chakras (theme duration)
Play each chakra for specified duration.
Parameter ‘theme’ specifies variant.

(fn THEME DURATION)

12.218.1.4 sox-relax

Command: sox-relax (duration-scale)
Play relax set of  binaural beats.
Each segment is scaled by ‘duration-scale’ in seconds.

(fn DURATION-SCALE)

12.218.1.5 sox-rev-up

Command: sox-rev-up (duration-scale)
Play rev-up set of  binaural beats.
Each segment is scaled by ‘duration-scale’ in seconds.

(fn DURATION-SCALE)

12.218.1.6 sox-tone-binaural

Command: sox-tone-binaural (length freq beat gain)
Play binaural audio with carrier frequency ‘freq’, beat ‘beat’,  and gain ‘gain’.

(fn LENGTH FREQ BEAT GAIN)

12.218.1.7 sox-wind-down

Command: sox-wind-down (duration-scale)
Play wind-down set of  binaural beats.
Each segment is scaled by ‘duration-scale’ in seconds.

(fn DURATION-SCALE)
